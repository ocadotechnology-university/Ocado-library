# Usage Guide for the Frontend

## Prerequisites

Before you begin, ensure you have the following installed on your system:

1. **Node.js** (v16 or later) - [Download and install Node.js](https://nodejs.org/)
2. **npm** (Node Package Manager) - Comes with Node.js installation.
3. **Vite** (optional) - Vite is used as the build tool for this project. It will be installed automatically as part of the dependencies.

## Installation

1. Clone the repository to your local machine:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the `library-ui` directory:
   ```bash
   cd /Users/uladzislautrybukhouski/Documents/PWR/Ocado-library/library-ui
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```

## Running the Application

To start the development server and run the application locally:

1. Run the following command:
   ```bash
   npm run dev
   ```

2. Open your browser and navigate to the URL provided in the terminal (usually `http://localhost:5173`).

## Testing the Application

To run the tests for the application:

1. Use the following command:
   ```bash
   npm test
   ```

2. The test results will be displayed in the terminal.

## Building for Production

To create a production build of the application:

1. Run the following command:
   ```bash
   npm run build
   ```

2. The production-ready files will be generated in the `dist` directory.

## Additional Notes

- Ensure that you have a stable internet connection to download dependencies during installation.
- If you encounter any issues, check the `README.md` file or the project documentation for troubleshooting steps.
